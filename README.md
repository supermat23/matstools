# 🎲 Mat's Tools

**Mat's Tools** est une application mobile/web tout-en-un conçue pour les joueurs de jeux de rôle et de jeux de société.

Elle regroupe plusieurs outils essentiels dans une seule application rapide, installable et utilisable hors ligne.

👉 Lancez vos dés, gérez vos scores et prenez vos notes sans quitter la table de jeu.

---

## 🧰 Outils inclus

### 🎲 Lanceur de Dés

* Dés classiques : D3 → D100
* Mode **simple**
* Mode **combat multi-joueurs**
* Pools de dés personnalisés
* Animation de lancer immersive
* Historique des lancers
* Tableau de probabilités

---

### 🏆 Compteurs de Scores

Parfait pour :

* Points de vie
* Scores joueurs
* Ressources
* Mana / Or / XP
* Jeux de plateau

Fonctionnalités :

✅ Compteurs illimités
✅ Organisation en grille / liste / cercle
✅ Modification rapide (+ / − / valeur personnalisée)
✅ Annulation des actions
✅ Sauvegarde de listes
✅ Import / Export JSON

---

### 📝 Notes intégrées

Un espace rapide pour :

* Notes de campagne JDR
* Rappels de règles
* Informations de scénario
* Organisation de parties

Fonctionnalités :

* Création rapide
* Recherche instantanée
* Couleurs personnalisées
* Sauvegarde et chargement

## 📱 Installation Mobile (PWA)

Mat's Tools fonctionne comme une **application native**.

### Android

1. Ouvrir dans Chrome
2. Menu ⋮
3. **Ajouter à l'écran d'accueil**

### iOS

1. Ouvrir dans Safari
2. Bouton **Partager**
3. **Sur l'écran d'accueil**

✅ Fonctionne hors ligne
✅ Plein écran
✅ Rapide et léger

---

## 💾 Données & Sauvegardes

Vous pouvez :

* Exporter vos données
